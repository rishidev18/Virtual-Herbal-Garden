# Virtual-Herbal-Garden
Virtual Herbal Garden 🌿 is an interactive web-based project that provides information about medicinal plants, including their scientific names, uses, health benefits, and cultivation methods. It promotes awareness of herbal medicine and biodiversity through an easy-to-use digital platform.

1. Project Overview:

The Virtual Herbal Garden is a web-based application designed to provide detailed information about medicinal plants and herbs. The system allows users to explore various herbal plants, learn about their scientific names, medicinal uses, health benefits, growing conditions, and native regions. The project aims to promote awareness of herbal medicine and preserve traditional knowledge through a digital platform.
